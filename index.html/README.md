# ⚖️ 正义之槌：AI 审判庭 (Justice's Gavel)

这是一个基于 React + Vite + Tailwind CSS 开发的沉浸式法官模拟决策游戏。玩家将扮演大法官，针对五个涉及 AI 伦理与法律的复杂案件作出裁决，并最终解锁不同的社会结局。

## 🎮 游戏特色
- **沉浸式体验**：深色木纹法庭设计，庄重权威。
* **动态动画**：每一次宣判都有震撼的法槌敲击效果。
* **多重结局**：根据你的判例倾向（技术导向 vs 人权导向），解锁 5 种不同的结局。
* **硬核案例**：涵盖 AI 招聘歧视、自动驾驶事故、医疗 AI 失误等前沿法律议题。

## 🚀 本地运行指南

如果你想在本地运行并体验此游戏，请按照以下步骤操作：

### 1. 环境要求
- [Node.js](https://nodejs.org/) (v18+)
- npm (v9+)

### 2. 安装与启动
1. **克隆项目**
   ```bash
   git clone https://github.com/你的用户名/你的仓库名.git
   cd 你的仓库名
   ```

2. **安装依赖库**
   ```bash
   npm install
   ```

3. **启动项目**
   ```bash
   npm run dev
   ```

4. **访问游戏**
   在浏览器中打开：`http://localhost:3000`

## 🛠️ 技术栈
- **框架**: React 19
- **构建工具**: Vite
- **样式**: Tailwind CSS 4
- **动画**: Framer Motion (motion/react)
- **图标**: Lucide React
